# DDUL V1 and VPCC repository
Home to DDUL license and VPCC along with the VPCC generator and documentation.
Get started by heading over to https://iaprofessional.github.io/ddul/

# Key Features
1) Human authored.
2) Version controlled standard.
3) Authoritative and stringent.
4) Remedies such as interim injunction and model disgorgement(destruction) available for breach of contract.
